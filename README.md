# EducationBI

A modern education analytics and reporting project built using Python, SQL, and Power BI to demonstrate how schools can transform raw operational data into meaningful insights for leadership teams, teachers, and student wellbeing staff.

## Project Overview

EducationBI simulates a real-world school analytics platform focused on:

* Student attendance tracking
* Academic performance monitoring
* Student engagement analytics
* Risk identification
* Executive reporting dashboards
* AI-generated student wellbeing summaries

The project combines data engineering, analytics, and visualisation workflows commonly used in modern education environments.

---

# Features

## Data Generation & ETL

Python notebooks generate realistic education datasets including:

* Students
* Courses
* Attendance
* Assignments
* Submissions
* Fees
* Enrolments
* Canvas LMS-style activity

These datasets are designed to simulate real school operational systems.

---

## Power BI Dashboards

Interactive dashboards include:

* Attendance trends
* Student risk profiles
* Engagement analytics
* Assessment performance
* Daily and weekly attendance monitoring
* At-risk student identification
* Executive summary KPIs

---

## AI Student Summaries

The project also explores AI-assisted reporting by generating automated student wellbeing and performance summaries using GPT APIs.

Example outputs include:

* Narrative wellbeing reporting
